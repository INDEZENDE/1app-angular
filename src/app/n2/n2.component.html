Tabla de tareas (ToDos) la cuál debe tener un formulario reactivo con un input para el nombre de la tarea y un botón para agregar cada tarea. Dichas tareas se guardarán en LocalStorage con nombre y fecha. La tabla debe mostrar un checkbox para completar la tarea, su nombre, la fecha de creación (con formato YYYY/MM/DD HH:MM) y un botón para eliminar individualmente cada tarea. Se deben mostrar por separado las tareas completadas de las que están por completar.
